SubTrack — 내 구독, 한눈에. 쉽게 관리.

📊 구독 자동 탐지
카드를 연동하면 결제 내역을 분석해 Netflix, Spotify, Adobe CC 등 구독 서비스를 자동으로 찾아드립니다.

🔔 D-3 결제 알림
결제일 3일 전에 미리 알림을 받아보세요. 모르고 지나치는 구독 요금을 방지할 수 있습니다.

📈 지출 리포트
월별 구독 지출 현황과 카테고리별 비중을 한눈에 확인하세요.

🔍 해지 안내
해지하고 싶은 구독이 있다면 단계별 가이드를 통해 쉽게 해지 페이지로 이동할 수 있습니다.

🔒 안전한 금융 데이터 처리
- 마이데이터 표준 API (금융위원회 인증) 활용
- AES-256 암호화로 금융 데이터 보호
- 실제 결제 기능 없음 — 조회 및 안내만 제공

【지원 카드사】

【개인정보 처리】
수집한 금융 정보는 구독 탐지 목적으로만 사용되며, 제3자에게 제공하지 않습니다.
자세한 내용은 앱 내 개인정보처리방침을 확인하세요.
